# 📝 Full-Stack Todo App

Hey there! Welcome to my full-stack Todo application. I built this project to practice building a complete, functioning web app and to keep daily tasks organized. 

To keep the development workflow professional, I've separated the architecture into different branches. You can easily explore the `frontend` and `backend` codebases independently using the branch dropdown above!

## 🛠️ Built With
* **Frontend:** React
* **Backend:** Node.js
* **Database:** PostgreSQL

## ✨ Features
* **CRUD Operations:** Easily create, view, edit, and delete tasks.
* **Responsive Design:** A clean and user-friendly interface.
* **RESTful API:** A standalone Node.js backend smoothly connected to a PostgreSQL database.
